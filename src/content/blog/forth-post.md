## Dzień 4: Pierwsze kroki w WordPress Development

### Podsumowanie dnia

🎯 Zrealizowane: 5,8 godziny nauki  
⏰ Łączny progres: 34,3h (14.3% miesięcznego celu)  
📚 Status: "Become a WordPress Developer: Unlocking Power With Code" (rozpoczęty)

### Kluczowe aspekty kursu

Rozpocząłem nową, ekscytującą podróż w świat WordPress Development. Kurs zapowiada się niezwykle obiecująco, oferując kompleksowe podejście do tworzenia zarówno klasycznych, jak i nowoczesnych rozwiązań w WordPress.

### Plan i zakres kursu

1. **Fundamenty rozwoju WordPress**

   - Tworzenie tematów klasycznych jako podstawa
   - Przejście do tematów blokowych jako naturalna ewolucja
   - Praktyczne wykorzystanie URL-based queries i custom queries
   - Praca z archives i loops

2. **Zaawansowane elementy**

   - Rozwój własnych wtyczek
   - Tworzenie stron akcji i ustawień
   - Integracja z BigData
   - Implementacja własnych tabel SQL

3. **Innowacyjne projekty**
   - Tworzenie wtyczek opartych na blokach
   - Implementacja interaktywnych quizów
   - Wykorzystanie najnowszych funkcji WordPress API
   - Budowa kompletnych rozwiązań dla uniwersytetów

### Środowisko developerskie

Kluczowym elementem rozpoczęcia pracy jest przygotowanie odpowiedniego środowiska deweloperskiego. Wymaga ono:

- PHP
- Apache/NGINX
- MySQL

Dostępne opcje konfiguracji:

- XAMPP
- MAMP
- AMPPS

Na potrzeby kursu wybrałem Local by Flywheel, które oferuje wygodne i kompleksowe rozwiązanie.

### Pierwsze kroki w PHP

Rozpoczęliśmy od podstaw PHP, poznając składnię i fundamentalne koncepcje:

```php
function greet($name, $author) {
    echo "<p>Hello, this is my $name function written by $author</p>";
}

greet("First", "Brad");
greet("Second", "John");
```
